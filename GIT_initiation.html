Logiciel de gestion de version¶

  • Il facilite le travail collaboratif
  • Stock l'historique des modifications
  • On peut trvailler sur plusieurs versions de prjets au meme temps
  • Possibilit de revenir en arriere
  • Permet de travailler sur le meme fichier par plusieurs personnes au meme temps

Exemple de logiciel¶

  • CVS
  • SVN
  • GIT (le plus utilis)
  • BAZAAR
  • MERCURIAL
  • ...

Pourquoi GIT est le meilleur ?¶

  • Il est cre par LINUS TORVALDS (createur de LINUX)
  • les developpeurs possedent tous la meme version avec tout l'historique (en local)
  • Disponibilits des depots GIT
  • GIT utilise peu de memoire par rapport aux autres VCS

Installation de GIT¶

  • https://gitforwindows.org/
  • Une version en ligne de commande (GIT bash), une version GUI (graphique)
  • De preference de commancer par ligne de commande

GitHub¶

  • https://github.com/account/unverified-email
  • rachet par microsoft
  • reseau social des developpeurs
  • permet de collaborer avec d'autres developpeurs
  • on peut creer des projets privs ou publics
  • c'est un depot parmis d'autres (GitLab, ...)
